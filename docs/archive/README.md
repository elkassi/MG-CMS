# Archive — historical plans & analyses

Everything in this folder is a **point-in-time** document: improvement plans, code reviews,
audit reports, and AI-session handoffs written between Dec 2025 and Jun 2026. Much of what
they propose has since been implemented, changed, or dropped.

**Do not implement from these documents.** For current guidance see [`docs/README.md`](../README.md).

| `plans/` | Large improvement plans (sequence dispatcher, master scheduling vision, testing guide), Apr – May 2026. Dispatcher/workbench plans are largely implemented. |
